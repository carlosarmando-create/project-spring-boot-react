# Frontend Plant Store

Frontend en Next.js + Tailwind CSS con:

- landing pública para clientes
- catálogo y contacto
- login por tipo de acceso
- dashboard administrativo
- dashboard de cliente

## Requisitos

- Node.js 20.9+ recomendado
- npm 10+ recomendado

## Variables de entorno

Copiar `.env.example` como `.env.local` y ajustar si hace falta.

## Ejecutar

```bash
npm install
npm run dev
```
